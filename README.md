# SwiftDL — Video & Audio Downloader

A **fully static** video downloader that works on **GitHub Pages** with zero configuration.
No server, no backend, no Node.js required.

## ✅ How to deploy on GitHub Pages (takes 2 minutes)

1. Create a new GitHub repository (or use your existing one)
2. Upload **only** the `index.html` file to the root of the repository
3. Go to **Settings → Pages → Branch: main → Folder: / (root) → Save**
4. Your app is live at `https://yourusername.github.io/repository-name/`

## How it works

| Feature | Powered by |
|---------|-----------|
| Video downloads | [Cobalt API](https://cobalt.tools) — free, no account needed |
| YouTube search & browse | [Invidious API](https://invidious.io) — open-source YouTube frontend |
| Video metadata | YouTube oEmbed + noembed.com |
| Download history | Browser localStorage |

## Supported sites

YouTube, TikTok, Instagram, Twitter/X, Facebook, Vimeo, SoundCloud, Dailymotion, Twitch and more.

## Features

- Paste any URL → see video info → choose quality & format → download
- Browse & search YouTube trending content
- Batch download multiple URLs
- Download history saved in browser
- Full blue & yellow theme with music instrument background

## No backend needed

Unlike the previous version, this app is **100% client-side**. Everything runs in your browser.
No Node.js, no Python, no yt-dlp to install.
